Big Idea

In this lab, we learned how to implement CSS files into our HTML, and how to create forms.

Challenges

The CSS file was already pre-created, so there were no issues regarding its implementation into our code. However, creating the form was more of a challange. With the introduction of new values and other coding options, new rules were established that had to be memorized. The main issue with the form was purely just correctly typing out the correct phrases combined with how time consuming it was to properly type them out.

results
